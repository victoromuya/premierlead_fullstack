Python for Web

Python is one of the very popular languages used in building web applications. Python has different frameworks which includes bottle.py, Flask, CherryPy, Pyramid, Django and web2py. These frameworks are been used to develop some of the most widely used apps in the world today. Systems like Spotify, Mozilla, Reddit, Netflix and so on are built with python framework.

What will you achieve at the end of the training?

  • Test Driven Application Development
  • Integrating back-end code with front-end frameworks
  • Deploying Python Web Application
  • Git and Github
  • Containerizing web applications
  • Authentications
